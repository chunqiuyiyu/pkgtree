>Show dependency tree of node project.

![preview](./preview.gif)

## Installation
```js
npm install pkgtree -g
```

## Usage
```shell
pkgtree [input]

Options:
  
  -l  show the tree with package version
  -v  output the version number
  -h  output usage information
```
